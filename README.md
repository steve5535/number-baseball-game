# number-baseball-game
숫자 야구 게임
숫자를 추측해 정답을 맞히는 간단한 게임이다.

## 기능
- 난이도에 따른 중복없는 랜덤한 수로 답을 생성한다.
  (쉬움:3개,중간:4개,어려움:5개)
- 사용자 입력에 따라 스트라이크 / 볼 / 홈런 / 아웃을 판단한다.
- 정답을 맞출 때까지 반복한다.
- 정답을 맞출 때까지 입력한 횟수를 카운트 해준다.

숫자 3자리를 입력해주세요 : 123
